# 🎓 Course Selling Application — MERN Stack

This project is a full-stack web application for selling online courses, built using the **MERN stack** (MongoDB, Express.js, React, Node.js).

## ✨ Features

- User registration and login (JWT authentication)
- Browse and search for courses
- Admin dashboard for course management
- Upload and manage course content (videos, PDFs)
- Order history and enrolled courses for users

## 🛠️ Tech Stack

- **Frontend:** React, TailwindCSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB (Mongoose ODM)
- **Authentication:** JWT & bcrypt
- **SechmaValidation:** ZOD
